# SKALA-FRONT 과제

HTML, CSS, JavaScript 강의자료의 지정 과제를 하나의 프로젝트로 통합한 결과물입니다.  
이 프로젝트는 포트폴리오가 아니라 **강의 과제 제출용**입니다.

## 실행 방법

1. 저장소를 클론합니다.

   ```bash
   git clone https://github.com/Heo-jinnyeong/skala-front.git
   ```

2. 프로젝트 폴더로 이동합니다.

   ```bash
   cd skala-front
   ```

3. VS Code에서 프로젝트 폴더를 엽니다.

4. Live Server 확장을 설치합니다.

5. 루트의 `index.html`을 열고 **Open with Live Server**를 실행합니다.
